# AMD GPU Architecture Programming Documentation

## Overview

A repository of AMD Instruction Set Architecture (ISA) and Micro Engine Scheduler (MES) firmware documentation.

## AMD MES Firmware

Documentation for the Micro Engine Scheduler firmware used in AMD GPUs.

## AMD GPU ISAs

### AMD RDNA Architecture
- RDNA (Radeon DNA) graphics processing unit (GPU) microarchitecture
- Successor to Graphics Core Next (GCN) microarchitecture
- Designed for gaming graphics products in the consumer market

### AMD CDNA Architecture
- CDNA (Compute DNA) architecture
- Designed specifically for data center and compute workloads
- High-performance computing focused

### Older Architectures
- AMD Vega
- AMD GCN 3

## AMD GPU Machine-Readable ISA Specifications

Machine-readable ISA specifications for AMD RDNA and AMD CDNA architectures. These are XML files that describe AMD's latest GPU instruction sets.

## Related Tools

### AMD Radeon GPU Analyzer (RGA)
Tool for analyzing and optimizing GPU code for AMD architectures.
