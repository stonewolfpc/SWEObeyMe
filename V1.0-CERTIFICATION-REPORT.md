## 11. Deployment Instructions

### For VS Code Marketplace
```bash
npm run build
vsce package
vsce publish
```

### For MCP Configuration
```json
{
  "mcpServers": {
    "sweobeyme": {
      "command": "node",
      "args": ["/path/to/dist/mcp/server.js"],
      "env": {
        "SWEOBEYME_MODE": "enterprise"
      }
    }
  }
}
```

### For Enterprise Deployment
1. Clone repository
2. Run `npm install`
3. Run `npm run build`
4. Configure MCP server in Windsurf/VS Code
5. Set enterprise mode: `SWEOBEYME_MODE=enterprise`
6. Enable audit logging: `SWEOBEYME_LOG_LEVEL=debug`
