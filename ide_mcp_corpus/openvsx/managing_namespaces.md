# Managing Namespaces

## Guidelines on Namespace Requests

This page provides guidelines for handling namespace-related requests, which are submitted as issues to [github.com/EclipseFdn/open-vsx.org](https://github.com/EclipseFdn/open-vsx.org/issues).

## Namespace Ownership

Namespaces are owned by the user or organization that created them. Ownership can be transferred with proper authorization.

## Renaming Namespaces

Currently, there is no explicit support to rename a namespace. You will need to create a new namespace and move your extensions to it.

## Deleting Namespaces

Namespaces can be deleted if they contain no extensions. If a namespace contains extensions, you must first delete or move all extensions before the namespace can be deleted.
