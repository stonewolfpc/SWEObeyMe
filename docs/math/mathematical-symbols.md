# Mathematical Symbols and Notation Reference

> **Purpose**: Quick reference for mathematical symbols used in programming and algorithms

## Set Theory

**∈** - Element of
```
x ∈ S  // x is an element of set S
```

**∉** - Not an element of
```
x ∉ S  // x is not an element of set S
```

**⊆** - Subset
```
A ⊆ B  // A is a subset of B
```

**∪** - Union
```
A ∪ B  // Union of sets A and B
```

**∩** - Intersection
```
A ∩ B  // Intersection of sets A and B
```

**∅** - Empty set
```
S = ∅  // S is the empty set
```

**|S|** - Cardinality
```
|S| = 5  // Set S has 5 elements
```

## Logic

**∀** - For all (universal quantifier)
```
∀x ∈ S, P(x)  // For all x in S, property P holds
```

**∃** - There exists (existential quantifier)
```
∃x ∈ S, P(x)  // There exists an x in S such that P holds
```

**¬** - Logical NOT
```
¬P  // Not P
```

**∧** - Logical AND
```
P ∧ Q  // P and Q
```

**∨** - Logical OR
```
P ∨ Q  // P or Q
```

**→** - Implies
```
P → Q  // P implies Q
```

**↔** - If and only if
```
P ↔ Q  // P if and only if Q
```

## Arithmetic

**∑** - Summation
```
∑(i=1 to n) a_i  // Sum of a_1 through a_n
```

**∏** - Product
```
∏(i=1 to n) a_i  // Product of a_1 through a_n
```

**√** - Square root
```
√x  // Square root of x
```

**|x|** - Absolute value
```
|x|  // Absolute value of x
```

**⌊x⌋** - Floor
```
⌊3.7⌋ = 3  // Floor of 3.7
```

**⌈x⌉** - Ceiling
```
⌈3.2⌉ = 4  // Ceiling of 3.2
```

## Calculus

**∫** - Integral
```
∫f(x)dx  // Integral of f(x) with respect to x
```

**∂** - Partial derivative
```
∂f/∂x  // Partial derivative of f with respect to x
```

**∇** - Gradient
```
∇f  // Gradient of f
```

**lim** - Limit
```
lim(x→a) f(x)  // Limit of f(x) as x approaches a
```

## Linear Algebra

**A** - Matrix
```
A = [a_ij]  // Matrix A with elements a_ij
```

**x** - Vector
```
x = [x_1, x_2, ..., x_n]  // Vector x
```

**Aᵀ** - Transpose
```
Aᵀ  // Transpose of matrix A
```

**A⁻¹** - Inverse
```
A⁻¹  // Inverse of matrix A
```

**|A|** - Determinant
```
|A|  // Determinant of matrix A
```

**||x||** - Norm
```
||x||  // Norm of vector x
```

## Probability

**P(A)** - Probability
```
P(A) = 0.5  // Probability of event A is 0.5
```

**E[X]** - Expected value
```
E[X]  // Expected value of random variable X
```

**Var(X)** - Variance
```
Var(X)  // Variance of random variable X
```

**σ** - Standard deviation
```
σ  // Standard deviation
```

**~** - Distributed as
```
X ~ N(μ, σ²)  // X is normally distributed
```

## Number Theory

**mod** - Modulo
```
a mod b  // a modulo b
```

**≡** - Congruent to
```
a ≡ b (mod n)  // a is congruent to b modulo n
```

**gcd** - Greatest common divisor
```
gcd(a, b)  // Greatest common divisor of a and b
```

**lcm** - Least common multiple
```
lcm(a, b)  // Least common multiple of a and b
```

## Programming-Specific Notation

**O(n)** - Big O notation
```
O(n)  // Upper bound of complexity
```

**Ω(n)** - Big Omega notation
```
Ω(n)  // Lower bound of complexity
```

**Θ(n)** - Big Theta notation
```
Θ(n)  // Tight bound of complexity
```

**:=** - Assignment
```
x := 5  // Assign 5 to x (pseudocode)
```

**≈** - Approximately equal
```
x ≈ y  // x is approximately equal to y
```

**∞** - Infinity
```
lim(x→∞)  // Limit as x approaches infinity
```

**∴** - Therefore
```
∴  // Therefore
```

**∵** - Because
```
∵  // Because
```

## Greek Letters Common in Programming

**α** - Alpha (learning rate, significance level)
**β** - Beta (coefficients, probability)
**γ** - Gamma (regularization parameter)
**δ** - Delta (change, Dirac delta)
**ε** - Epsilon (small value, error term)
**θ** - Theta (parameters, angle)
**λ** - Lambda (regularization parameter, eigenvalue)
**μ** - Mu (mean)
**σ** - Sigma (standard deviation, sum)
**π** - Pi (constant, 3.14159...)
**φ** - Phi (golden ratio, angle)
**ω** - Omega (angular frequency)
